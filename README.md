## 📌 About

이 프로젝트는 AWS 기반으로 웹 서비스를 배포하고,  
**자동 배포(CI/CD)와 CDN을 적용한 클라우드 아키텍처를 직접 구축**한 프로젝트입니다.

초기에는 EC2 + Nginx 방식으로 웹 서버를 구성하였고,  
이후 S3 + CloudFront 구조로 확장하여  
**정적 웹 서비스 + HTTPS + 자동 배포 환경**까지 구현했습니다.

---

## ⚙️ Features

- EC2 + Nginx 웹 서버 구축
- S3 정적 웹 호스팅
- CloudFront CDN + HTTPS 적용
- GitHub Actions 자동 배포
- 도메인 연결 (www / root 처리)

---

## 🏗 Architecture

GitHub → GitHub Actions → S3 → CloudFront → User


---

## 🧪 Usage

1. index.html 수정  
2. GitHub commit  
3. 자동 배포 실행  
4. 사이트 반영 확인
